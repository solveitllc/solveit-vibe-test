# SolveIT - Vibe Coder Test

## Instrucciones

Este es un mini proyecto Next.js con bugs intencionales. Tu tarea:

1. **Clona este repo**
2. **Crea un branch** llamado `fix/test-[tunombre]`
3. **Encuentra y corrige los bugs** en `src/components/ClientList.jsx`
   - Puedes usar cualquier herramienta de AI (Cursor, Copilot, Claude, ChatGPT, etc.)
   - El componente tiene **3 bugs** — encuéntralos todos
4. **Haz commit** con mensajes descriptivos y push a tu branch

## Setup

```bash
git clone <repo-url>
cd solveit-vibe-test
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Tiempo estimado

~15 minutos

---

> **Nota:** Este test evalúa tu capacidad de debuggear con AI, no de memorizar sintaxis.
